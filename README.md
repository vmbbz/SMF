# Solana Meme Fighter

Web3 meme token gamified fighting game where players battle against trending Solana Pump.fun tokens.

## Features

### Meme Token Integration
- **Trending Arena**: Auto-load hottest Pump.fun tokens as opponents
- **Custom Fights**: Paste any Solana mint address for instant 1v1
- **Endless Pump Stream**: Non-stop meme opponents with 30-second refresh
- **Market Scaling**: Opponent difficulty scales with real trading volume
- **Token Logos**: Display as fighter heads with fallback circles
- **Dynamic Personalities**: Each token gets unique taunts and voice traits
- **Victory Screenshots**: Auto-capture wins with one-click X posts
- **Browser Voice Taunts**: Zero-cost TTS for trash talk
- **PWA Ready**: "Add to Home Screen" on mobile devices

### Combat System
- **6 attack types**: light/medium/heavy punch and kick
- **Hit zones**: head (2x), crotch (3x), body (1x), limbs (0.5x)
- **Mechanics**: blocking, dashing, double-jumping, somersaults

### Tech Stack
- **Frontend**: Pure HTML/JS with Canvas rendering
- **Token APIs**: Dexscreener (trending), Birdeye (token metadata)
- **Voice**: Browser Web Speech API for taunts
- **Deployment**: Static hosting ready

## Quick Start

1. Open `index.html` in browser
2. Click "Trending Meme Arena" or paste token address
3. Battle meme tokens with karate moves
4. Win → screenshot + X post automatically

## Controls

### Keyboard (P1)
| Keys | Action |
|------|--------|
| W A S D | Move (up/left/down/right) |
| U I O | Light / medium / heavy punch |
| J K L | Light / medium / heavy kick |
| Double-tap A or D | Dash |
| W W (double-tap) | Somersault |

### Keyboard (P2)

| Keys | Action |
|------|--------|
| Arrow keys | Move |
| 4 5 6 (numpad) | Light / medium / heavy punch |
| 1 2 3 (numpad) | Light / medium / heavy kick |

### Voice / Phone Commands

Shout any combination: `"forward"`, `"jump"`, `"heavy kick"`, `"dash forward"`, `"crouch heavy punch"`, `"jump forward heavy kick"`, etc.
